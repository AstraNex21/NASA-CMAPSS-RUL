# NASA C-MAPSS Remaining Useful Life Prediction

## Objective

Develop a machine learning pipeline for Remaining Useful Life (RUL) prediction on the NASA C-MAPSS turbofan engine dataset using explainable AI (SHAP) for sensor selection.

## Research Goals

- Build a robust preprocessing pipeline.
- Develop baseline RUL prediction models.
- Rank sensor importance using SHAP.
- Compare model performance before and after feature selection.
- Produce publication-quality figures and tables.
